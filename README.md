# 🌐 Vivekananthan T — Portfolio Website

A fully responsive personal portfolio and resume website built with **React.js**, showcasing my projects, certifications, internships, and skills.

🔗 **Live Site:** ([https://vivekananthant06.vercel.app])

---

## 📌 About

This is my personal portfolio website that I built to present my work and background as a Computer Science and Engineering student at Dhanalakshmi Srinivasan Engineering College.

It includes 4 pages:
- **About Me** — Bio, technical skills, soft skills, and contact info
- **Projects** — Medoo+ AI-powered healthcare platform with live demo and GitHub links
- **Certifications** — Internships, course certifications, workshops, and education history
- **Resume** — Full resume preview with a downloadable PDF

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/vivekananthan-t/portfolio.git

# Navigate into the folder
cd portfolio

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
